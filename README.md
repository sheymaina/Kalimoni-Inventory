 Kalimoni Inventory Console

An inventory management system using Object-Oriented Programming. This project helps large/mega shops break a big problem into objects with clear responsibilities.


### Features
- Product Management: Track name, price, stock levels, and auto-calculate stock value
- Customer System: Regular customers + VIP customers with discount rates
- Order Processing: Automatically reduces stock and updates purchase history
- Stock Control: Prevents overselling, supports restocking
- Inheritance: VIPCustomer inherits from Customer and overrides total_spent()
- Reporting: View full system state at any time
